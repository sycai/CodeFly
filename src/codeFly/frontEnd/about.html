What can CodeFly do? It'll blow your mind.

CodeFly is a web application aims to assist programmers to manage, compile and test their code remotely.

Editing code in a web browser.

Uploading code files in batch to the server.

Downloading code files in batch from the server.

Compiling and running code in the server environment.

Remotely testing the code.

Why CodeFly ?.

Algorithm questions in the technical interviews block numerous talented programmers from landing in their dream jobs. However, it is also well known that an interviewee would find these questions less formidable after practice.

CodeFly is designed to provide fast and stable accesses for programmers who want to practice their algorithm skills online.
